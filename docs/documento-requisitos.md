## Projeto Cordel Moderno - Documento de Requisitos

## Descrição

Este projeto foi baseado inicialmente no projeto desenvolvido pelo **Curso em Vídeo**, ministrado pelo professor **Gustavo Guanabara**, no módulo 3 do curso de HTML5 e CSS3.

A proposta é criar uma página web apresentando uma poesia de cordel brasileira, utilizando imagens de fundo com efeito **parallax**, buscando valorizar e homenagear essa expressão cultural.

O projeto receberá modificações pessoais, com o objetivo de aprimorar o design, estrutura e experiência do usuário.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* Visual Studio Code (IDE)

---

## Requisitos funcionais

* Criar uma página web contendo uma poesia de cordel.
* Utilizar imagens de fundo com efeito parallax.
* Organizar o conteúdo utilizando HTML semântico.
* Criar uma estrutura de página contendo:

  * Cabeçalho (Header)
  * Conteúdo principal (Main)
  * Rodapé (Footer)
* Adicionar elementos visuais relacionados à cultura do cordel.

---

## Requisitos não funcionais

* O projeto deve possuir uma estrutura organizada.
* O layout deve se adaptar a diferentes tamanhos de tela.
* O código deve seguir boas práticas de HTML e CSS.
* O design deve manter uma identidade visual inspirada no cordel.
