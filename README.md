# 🌵 Projeto Cordel Moderno

Projeto desenvolvido durante o **Módulo 03 do curso de HTML5 e CSS3** do Curso em video

O objetivo do projeto foi criar uma página web responsiva utilizando os fundamentos de **HTML5 e CSS3**, aplicando conceitos de estruturação semântica, organização de código e adaptação para diferentes tamanhos de tela, e utilizar o efeito parallax em imagens de fundo.

- 🔗 Acesse o projeto: [Link do projeto](https://devgustavojeferson.github.io/Projeto-Cordel-Moderno/)

## 🎯 Objetivo do Projeto

Criar uma página poética sobre a cultura de cordel, praticando:

- Estruturação de páginas com HTML5
- Estilização com CSS3
- Organização de arquivos
- Responsividade
- Boas práticas de desenvolvimento
- Efeito Parallax
